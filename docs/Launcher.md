# Launcher

## 项目介绍

程序启动器，通过 INI 配置文件批量启动程序

## 使用说明

### 运行方式

- 直接运行程序，会自动查找与程序同名的 INI 文件（例如程序为`Launcher.exe`，则查找`Launcher.ini`）
- 若未找到配置文件，程序会自动创建示例 INI 文件并提示用户

### INI 配置文件格式

每个程序配置以`[节名]`区分，节内可设置以下参数：

| 参数名       | 说明                                                         | 示例                                                     |
| ------------ | ------------------------------------------------------------ | -------------------------------------------------------- |
| `Program`    | **必填**，程序路径（支持绝对路径、相对路径或系统 PATH 中的程序名） | `notepad.exe`、`./app.exe`、`C:\tools\tool.exe`          |
| `Params`     | 可选，程序启动参数                                           | `param1 --mode=debug`                                    |
| `WorkDir`    | 可选，程序工作目录（默认使用程序所在目录）                   | `./data`、`D:\workspace`                                 |
| `RunAsAdmin` | 可选，是否以管理员身份运行（`1`= 是，`0`= 否，默认`0`）      | `1`                                                      |
| `ShowWindow` | 可选，窗口显示方式（默认`SW_NORMAL`，即正常显示）            | 可取值：`SW_HIDE=0`（隐藏）、`SW_MAXIMIZE`=3（最大化）等 |
