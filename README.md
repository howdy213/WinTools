# WinTools

## 项目简介

各种实用工具的集合

## 包含项目

* [AutoStartService](docs/AutoStartService.md) 自启服务
* [Flake](docs/Flake.md) 模拟息屏
* [HotspotHelper](docs/HotspotHelper.md) 热点管理器
* [Launcher](docs/Launcher.md) 程序启动器
* [LightPanel](docs/LightPanel.md) 自带UIAccess的小面板
* [ShutdownEx](docs/ShutdownEx.md) 关机工具
* [TaskManager](docs/TaskManager.md) 任务计划程序

## 环境依赖

Visual Studio 2022

## 项目依赖

[WinUtils](https://github.com/howdy213/WinUtils)(Windows 微功能组件库)
